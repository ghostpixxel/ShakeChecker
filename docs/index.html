Get started

Running in under a minute

  1. Download the latest release and extract it anywhere.
  2. Run ShakeChecker.exe. It lives in your system tray.
  3. First launch only: if Windows shows "Windows protected your PC", click More info → Run anyway (the app is unsigned).
  4. Play PokeMMO in windowed or borderless mode (English client). The overlay shows in battles.

Ban-safe by design

ShakeChecker is built read-only on purpose. It only reads your screen and draws an overlay, it never sends inputs, never reads game memory, and never modifies the client. There is nothing for it to automate; you still play every battle yourself.

FAQ

Questions

Using it
The overlay doesn't show up.
Run PokeMMO in windowed or borderless mode (not exclusive fullscreen), use the English client, and start a wild battle, the overlay is hidden in the overworld and in trainer battles.
The turn counter is wrong or not updating.
Keep the in-game chat visible with the Battle tab active. ShakeChecker reads "Turn N started!" from the battle chat, and corrections kick in from turn 2 onward.
It didn't recognise the Pokémon or its status.
The name and status are read from the screen, so keep them visible and unobscured by animations. Sleep, paralysis, poison, burn and freeze are all recognised.
Location Dex
What is the Location Dex?
It reads your current location from the HUD and lists the Pokémon that spawn on that route, minus the ones you've already caught, so you instantly see what you still need where you're standing.
How does it know what I've already caught?
It detects your OT catches in battle (the red Poké Ball icon by the enemy's name) and records them per account. You can also click any entry to mark it caught or uncaught yourself.
Does it account for time of day and season?
Yes. It factors in the current period and season, and shows each Pokémon's rarity (by colour) and how it's found (grass, water, fishing, lure, and so on).
Safety
Is it safe to use? Will I get banned?
It's built to be ban-safe: strictly read-only, it never sends inputs, never reads game memory and never touches the client. It only reads the screen and draws an overlay.
Is it a bot? Does it automate anything?
No. It never presses keys or clicks into the game. You play and throw balls yourself.
Why does Windows warn me on first launch?
The app is unsigned (signing certificates are expensive for a free tool), so SmartScreen warns once. Click More info → Run anyway. The full source is on GitHub if you'd rather build it yourself.
Does it use the internet or collect data?
No. It makes no network connections, and your caught lists are stored locally in %APPDATA%\ShakeChecker.
Good to know
Which catch formula does it use?
PokeMMO's Gen 3/4 catch formula, with the game's own ball and status multipliers.
Multiple accounts?
Yes, caught lists are kept separately per account.
Is it heavy on my PC?
No, it runs at about 5 frames per second during battles and 2 while walking.
What isn't supported yet?
The Safari Zone (it uses a different catch mechanic), Alpha Pokémon rates, and non-English clients. These are on the roadmap.
